# Machine Learning Portfolio (Python)

This repository contains a collection of machine learning projects focused on classification and regression problems. Each project includes data preprocessing, model training, evaluation, and visualization.

## 🚀 Projects

### 1. Credit Card Fraud Detection

* Binary classification on highly imbalanced data
* Model: SGDClassifier (logistic regression) with class balancing
* Metrics: ROC AUC, PR AUC, confusion matrix
* Key focus: handling class imbalance and threshold tuning

### 2. Heart Disease Prediction

* Classification problem using medical dataset
* Model: XGBoost with GridSearchCV
* Techniques: feature engineering, OneHotEncoding, hyperparameter tuning
* Optimization target: recall (important for medical diagnosis)

### 3. Laptop Price Prediction

* Regression problem
* Model: ElasticNet with hyperparameter tuning
* Advanced preprocessing:

  * feature extraction (CPU, resolution, memory)
  * target encoding
* Metrics: R² score, MAE
* Includes feature importance analysis

## 🛠️ Technologies

* Python
* scikit-learn
* XGBoost
* pandas, numpy
* matplotlib, seaborn

## 📌 Key Skills Demonstrated

* Data preprocessing and feature engineering
* Handling imbalanced datasets
* Model selection and evaluation
* Hyperparameter tuning (GridSearchCV)
* Building ML pipelines
